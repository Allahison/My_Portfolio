# Muhammad Arslan — Portfolio

Personal portfolio website built with Next.js, TypeScript, and Tailwind CSS.

**Live:** [my-portfolio-seven-blond-geucbic1wb.vercel.app](https://my-portfolio-seven-blond-geucbic1wb.vercel.app)

## Tech Stack

- Next.js 16
- TypeScript
- Tailwind CSS
- Deployed on Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view locally.
